# 🌾 AI Chatbot for Farmers

An intelligent agricultural assistant powered by Retrieval-Augmented Generation (RAG), Google Gemini, LangChain, and Pinecone. The platform enables farmers, students, and agricultural professionals to interact with domain-specific knowledge through natural language conversations.

The chatbot retrieves information from curated agricultural datasets and documents to provide contextual guidance on crop cultivation, fertilizers, irrigation techniques, pest management, disease prevention, and sustainable farming practices.

## 🚀 Key Features

* Agricultural knowledge retrieval using RAG architecture
* Natural language question answering
* Context-aware responses powered by Google Gemini
* Semantic search with Pinecone vector database
* PDF and document-based knowledge ingestion
* Interactive Streamlit web interface
* Scalable knowledge base for agricultural applications
