# 🌾 RAG Chatbot for Farmers

An AI-powered agricultural assistant built using Retrieval-Augmented Generation (RAG), Google Gemini, LangChain, Pinecone, and Streamlit.

The chatbot helps farmers access crop information, farming best practices, disease management techniques, fertilizer recommendations, and agricultural knowledge through natural language conversations.

## Features

- Agricultural knowledge retrieval
- PDF-based knowledge base
- Gemini-powered responses
- Pinecone vector search
- Streamlit web interface
- Context-aware answers
- Multi-document support

## Tech Stack

- Python
- LangChain
- Google Gemini
- Pinecone
- Streamlit
